# agent-skills

My personal Claude Code agent skills.

## Skills

- **yeet** — Fast commit-and-push. Usage: `/yeet "message"` or `/yeet` to auto-generate.
- **codex** — Dispatch coding tasks to OpenAI's Codex CLI as a subagent. Usage: `/codex "fix the failing tests"` or ask to "run this in codex".

## Installation

```
/plugin marketplace add GeorgeNance/agent-skills
/plugin install shortcuts@george-skills
```
